Politique de Confidentialité

Protection des données

Nous respectons les données des visiteurs de ce site. Lorsque vous visitez le site Web cnc-proto.eu, vous n’avez pas à fournir d’informations personnelles pour prendre connaissance des informations qu’il contient.

Si toutefois vous utilisez le formulaire de demande de devis sur ce site Web, les informations que vous nous envoyez sont traitées de manière confidentielle et conservées aussi longtemps que nécessaire, en fonction de la nature de votre question, de votre commentaire ou de votre demande.

Si vous passez une commande chez nous, une fois la commande traitée, vos coordonnées et les fichiers téléchargés seront stockés dans notre base de données clients pour les commandes ultérieures. Vous avez le droit à tout moment de demander un aperçu de ces informations, de les rectifier et/ou de les effacer. Dans ce cas, veuillez s’il vous plaît nous contacter.

Cookies

Ce site n’utilise pas de cookies.

Comportement et données des visiteurs

Alors que nous utilisions auparavant Google Analytics sur notre site Web, nous n’avons jamais examiné les données recueillies. Au moment où cnc-proto.eu a été lancé, c’était l’outil de choix à avoir.

Cependant, nous avons réalisé que nous ne nous soucions pas de ces données et les avons trouvées non pertinentes pour notre mission.

La morale de l’histoire est:

Ce n’est pas parce que vous pouvez obtenir des données que vous devriez le faire.

Nos nouvelles pages n’incluent aucun outil d’analyse ni aucune technologie de suivi. Par défaut, comme le font la plupart des serveurs Web, le serveur sur lequel ce site Web est stocké enregistre les visites. Dans la base de données du serveur Web, les informations suivantes sont enregistrées :

  • Votre adresse IP publique;
  • La date et l’heure de votre visite;
  • Les pages que vous avez visitées.

Tiers

Ce site Web est hébergé par Webhs, une société d’hébergement portugaise. De plus, si vous choisissez d’utiliser le formulaire de demande de devis disponible sur ce site, vous reconnaissez que les données que vous saisirez seront traitées par Cognito Forms avant de nous être transmises. Si vous n’êtes pas d’accord avec cela, veuillez nous envoyer votre demande de devis directement en utilisant notre adresse email.

À l’exception des cas ci-dessus, vos données ne sont pas mises à la disposition de tiers, sauf si cela est inévitable en raison de la nature du support de communication (par exemple, nos e-mails sont gérés par Google Workspace), pour le traitement des commandes (par exemple, nos factures sont traitées à l’aide du logiciel fourni par Moloni) ou à des fins fiscales.
